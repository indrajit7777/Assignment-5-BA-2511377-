# Part 3: NLP and Sequence Modeling Mini Project

*## Completed Tasks
- Task 1: Dataset understanding & EDA  
- Task 2: Text preprocessing (lowercasing, cleaning, tokenization, stopword removal)  
- Task 3: Text vectorization (Bag of Words, TF-IDF, Tokenizer + padded sequences)  
- Task 4: Baseline model (Logistic Regression + TF-IDF) with full evaluation  
- Task 5: Sequence model (LSTM)  
- Task 6: Reflection on RNNs, LSTMs, Attention & Transformers  

## How to Run
1. Download `customer_support_text_classification.csv` from the Google Drive link.
2. Place it in this folder.
3. `pip install -r requirements.txt`
4. Open `notebook.ipynb` and run all cells.
5. Check the `results/` folder.

**Files in `results/` folder (generated automatically):**
- `model_evaluation.csv` (metrics)
- `sample_predictions.txt`
- `class_distribution.png`
- `confusion_matrix.png`
